PRAPARE Survey

Thursday, Nov 19, 2020

PRAPARE SURVEY - This is an open-source work in progress. The Protocol for Responding to and Assessing Patients' Assets, Risks, and Experiences (PRAPARE) survey was recreated in a Flutter environment using its [LOINC FHIR® specification]. The app directly loads the FHIR® spec (albeit with some basic modifications to the original FHIR® JSON to more closely match the initial survey). It has basic validation in place for survey completion. Features Personal Characteristics Personal Characteristics, filled out Social and Emotional Health

AMIA Symptom Tracker

Thursday, Nov 19, 2020

AMIA Symptom Tracker - This was the demo application used for live-coding in the 4-hour Flutter workshop given at AMIA’s 2020 Virtual Clinical Informatics Conference. A total of 6 informaticians hosted this workshop. Features Welcome Screen, live-coded in full Symptom Tracker Dashboard, live-coded in full Check-In Screen, live-coded in full Firebase Puns, created by audience using Live Share Flutter / Firebase connection, previously created while live-coding FHIR® Package Demo, creating live patients on HAPI and Aidbox FHIR servers.
